## 三、详细阶段规划

### 第一阶段：大一暑假（2026年6-8月）

#### RE 专项（每日 1h）

```
第 1-2 周：汇编基础
  ├─ 资源：CSAPP 第3章、bilibili "x86汇编入门"
  ├─ 练习：写C代码 → gcc -S 看汇编 → 逐行对照
  └─ 目标：能读懂 if/for/while/switch 的汇编对应

第 3-4 周：工具上手
  ├─ Kali 虚拟机里装 Ghidra、GDB+pwndbg、pwntools
  ├─ 导入第一个 Crackme
  └─ 跟着 bilibili 教程做第一个逆向题目

第 5-8 周：Crackmes 刷题
  ├─ 从 2-3 分题开始
  ├─ 目标：独立完成 10+ 道题
  └─ 每题写 writeup 记录思路
```

#### ACM 专项（每日 3h）

```
专题突破：
  ├─ DP：区间DP、树形DP、状压DP、概率DP
  ├─ 图论：最短路径、拓扑序、生成树、二分图
  ├─ 数据结构：线段树变种、树状数组、ST表
  └─ 每周至少 1 场 CF 模拟赛（5h）
```

#### 网安基础（每日 0.5h）

```
第 1-2 周：计算机网络
  ├─ 资源：bilibili 中科大郑烇《计算机网络》
  └─ 重点：TCP/IP、HTTP/HTTPS、DNS

第 3-4 周：Linux 基础
  ├─ 资源：《鸟哥的 Linux 私房菜》基础篇
  └─ 重点：命令行、文件权限、进程管理、Shell 脚本

第 5-8 周：安全概念入门
  ├─ 资源：bilibili "CTF入门" / "Web安全入门"
  └─ 重点：OWASP Top 10、SQL注入、XSS、文件上传
```

## 五、工具与环境

### RE 工具

| 工具 | 平台 | 用途 |
|------|------|------|
| Ghidra | Linux/Windows | 主力静态分析（免费，反编译质量好） |
| IDA Free/Pro | Linux/Windows | 备选静态分析（F5反编译更接近源码感） |
| GDB + pwndbg | Linux | 动态调试 ELF |
| x64dbg / x32dbg | Windows | 动态调试 PE |
| pwntools | Python | 写解题脚本 |
| Z3 / angr | Python | 约束求解、符号执行 |
| Detect It Easy | Windows | 查壳、编译器识别 |

**Kali 安装命令：**
```bash
sudo apt install ghidra gdb python3-pip
pip install pwntools z3-solver
git clone https://github.com/pwndbg/pwndbg
cd pwndbg && ./setup.sh
```

### 网安工具（Kali 内置）

| 工具 | 用途 |
|------|------|
| nmap | 端口扫描 |
| Wireshark | 流量分析 |
| Burp Suite | Web渗透核心 |
| sqlmap | SQL注入自动化 |
| Metasploit | 渗透测试框架 |
| John the Ripper / Hashcat | 密码破解 |

## 十二、给你的核心建议

1. **ACM 有时效性，RE 没有** — 大二全力冲 ACM 铜牌，大三大四再深钻 RE
2. **安全行业不看学校牌子** — 看你能不能干活，竞赛成绩能帮你穿过滤网
3. **暑假先搞定 RE 入门** — 汇编 + 工具 + 10道 Crackmes，建立基础
4. **寒假突击网安基础** — Kali + Web 安全 + 靶场，打开攻击者视角
5. **写好 Writeup** — 做题的过程记录下来，既是复习，也是简历资产
6. **ACM 打出成绩后别放 AC 完全冷掉** — 保持 CF 手感，大三大四还能打邀请赛
